# cli-bmi

A CLI tool for BMI calculation.

## Install

```sh
npm i -g cli-bmi
```

## Usage

```sh
bmi [<options>]
bmi
bmi --weight 85 --height 180
```

## Commands

```
--height number   Height in centimeters, e.g. 180
--weight number   Weight in kilos, e.g. 85
-c, --clear       Clear local data
-h, --help        Show help section
```
